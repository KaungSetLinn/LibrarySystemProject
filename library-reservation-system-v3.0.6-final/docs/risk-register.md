## v3.0.3 で追加されたリスク・将来計画

### R-11 SQLite ブラウザモード非実体化（S-1 / KL-08）
- **影響度**: 中
- **対応状況**: v3.0.3 で起動時警告を追加、文書化済
- **v3.1 計画**: WASM SQLite の検討、または server/ モード必須化のドキュメント整備

### R-12 簡易認証の脆弱性（A-6 / KL-05）
- **影響度**: 高（本番運用時）
- **対応状況**: v3.0.3 で `users.password_hash` 列追加、bcryptjs 依存登録、文書化済
- **v3.1 計画**: EX-01 実装（bcrypt.compare 実装、ログイン画面パスワード入力欄追加）

### R-13 タブ間排他制御の限界（A-8 / KL-07）
- **影響度**: 中（ブラウザモードのみ）
- **対応状況**: 文書化済。server/ モードでは UNIQUE INDEX で物理排除
- **v3.1 計画**: Web Locks API または IndexedDB トランザクションへの移行

### R-14 ExcelAdapter の巨大化（C-13 / KL-09）
- **影響度**: 低（保守性のみ）
- **対応状況**: 文書化のみ
- **v3.1 計画**: モジュール分割（5ファイル構成）

### R-15 設定ファイル二重管理（C-15 / KL-10）
- **影響度**: 低（運用ミス）
- **対応状況**: 整合性チェック警告あり
- **v3.1 計画**: ビルド時 txt→js 自動生成スクリプト導入
